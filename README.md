\# DeFi Token Project



An ERC20 token implementation with minting and burning capabilities.



\## Features



\- ERC20 standard compliance

\- Token minting (owner only)

\- Token burning

\- Maximum supply cap

\- Built with Hardhat and OpenZeppelin
